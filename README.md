## Quick start
- In `adventurers` directory `cargo run path/to/map.ron q[i]`, `i = 1 | 2 | 3`
- See `adventurers/src/main.rs` and `adventurers_quest/src/lib.rs` to add new quests to the game
- See `maps` to add new maps to the game
